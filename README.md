## 🛠️ Getting Started

To run this project locally, follow the steps below:

```bash

git clone https://github.com/Ishavanjare/My_Portfolio.git

# Navigate to the project directory
cd My_Portfolio

# Install dependencies
npm install

# Start the development server
npm start
